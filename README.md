# credit-card-design-backend

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:7000
```

### MongoDB

configure .env file with your MongoDB URI, local or Atlas

## Usage

On your postman input
http://localhost:7000/api/signup
